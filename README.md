# student_performance_manager_version-2.py
This is the second version of student performance manager
The Student Performance Manager is a Python command-line application designed to help manage student academic performance.

This is the second version of the project, showing improvements in program logic and feature implementation compared to the first version.

The application is inspired by the structure of the British education system.
This project was built to practice:

Python programming fundamentals

Data management using lists

Conditional logic and loops

Basic student performance analysis

It is a learning project aimed at improving coding skills.
